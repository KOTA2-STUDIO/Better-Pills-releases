これは何？

Better Pills は、8 個のステップに設定した Wah 位置（周波数）を、テンポに同期して順に切り替えるステップシーケンサー型の Wah です。 STEPS で 4 / 6 / 8 ステップ、SYNC で DAW テンポシンク / タップテンポ、ORDER で Forward / Random を選べます。 RATE / RESONANCE / RANGE / GLIDE / MIX で質感を追い込めます。

インストール

macOS

ダウンロードした .dmg を開き、中の .pkg を実行すると、Audio Unit（Logic / GarageBand 用）と VST3（その他の DAW 用）が インストールされます。完了後、DAW でプラグインを再スキャンしてください。

Windows

ZIP を解凍し、Better Pills.vst3 フォルダを C:\Program Files\Common Files\VST3\ にコピーしてください（管理者権限が必要）。完了後、DAW で再スキャンします。

動作環境

macOS（Universal: Apple Silicon / Intel）・Windows 10 / 11 (64-bit)。
AU または VST3 対応の DAW。mono / stereo 対応。
