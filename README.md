# Better Pill — Releases & Download

[Better Pill](https://github.com/KOTA2-STUDIO/Better-Pill) のダウンロードページと
配布バイナリを公開するリポジトリです。ソースコードは別の Private リポジトリにあります。

## ダウンロード

👉 **https://kota2-studio.github.io/Better-Pill-releases/**

または [Releases](https://github.com/KOTA2-STUDIO/Better-Pill-releases/releases/latest) から直接：

| OS | ファイル | 永続リンク（最新版） |
|----|---------|----------------------|
| macOS | `.pkg` / `.dmg` | `releases/latest/download/Better-Pill-macOS.dmg` |
| Windows | VST3 ZIP | `releases/latest/download/Better-Pill-Windows-VST3.zip` |

各リリースには、バージョン入りのファイル（アーカイブ用）と、バージョンなしの
固定名エイリアス（`latest/download` の永続リンク用）の両方を置いています。

## 構成

- `docs/` — ランディングページ（GitHub Pages）
- Releases — 配布バイナリ。ソースリポジトリの CI から自動転載されます。
